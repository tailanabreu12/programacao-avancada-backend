## Objetivo

Se aprofundar no desenvolvimento backend de sistemas com JavaScript, trabalhando com NodeJs

<!-- ![](header.png) -->

## Instalação

Gerar arquivo configuração do package.json

```
npm init -y

```

Instação do nodemon

```
npm i nodemon -D
```

```json
"scripts": {
    "start": "nodemon server.js"
  },
```
